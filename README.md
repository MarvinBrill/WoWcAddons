# WoWcAddons

A collection of World of Warcraft Classic Addons.

## Addons

### TimeUntilLevelUp
Tracks your leveling progress by calculating time and XP chunks remaining until the next level.

**Features:**
- **XP Chunks**: Shows how many more "actions" (based on last XP gain) are needed to level up.
- **Leveling Timer**: Tracks XP/Hour and estimates time remaining.
- **Movable UI**: Dark, transparent window that can be positioned anywhere.

**Usage:**
- The window appears automatically.
- Click "Start Timer" to begin tracking session XP statistics.
